# Event Attendance Management System

This project automates attendance tracking using a real-time data flow system.

## 🔄 How it works

1. User fills Google Form  
2. Data is sent to backend  
3. Backend processes and stores in database  
4. Frontend displays attendance in real-time  

## 📌 Features

- Automated attendance system  
- Real-time updates  
- Database storage  
- Dashboard visualization
